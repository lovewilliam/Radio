Hardware Prototype

WaveRadio is a built-from-the-ground USB radio. It also contains instructions on how to build your own USB radio with a fancy front GUI.

WaveRadio is currently available on Linux and MeeGo devices. Ports for other platforms are under development.

Hardware and Firmware

The hardware is built with an ATMega 8 and a TEA5767 module.
For Schematics See Here .
This is Firmware .


Tuner for PC

FreqControl is a command line program which can help control the radio.
